Ques: 5 Write a program that takes a sentence as user input and capitalizes the text in a specific way based on the number of words. The conditions are as follows: If the number of words in the sentence is less than 4, capitalize the first letter of each word. If the number of words is 4 or more, only capitalize the first letter of the first word. The program should display the modified sentence on the DOM as an output. Explore toUpperCase(), charAt() and slice() methods for this exercise.
